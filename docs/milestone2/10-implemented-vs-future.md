# Implemented vs future

Use [11-implemented-vs-future.md](11-implemented-vs-future.md) and [AI_SYSTEM_README.md](AI_SYSTEM_README.md) as the current capability summaries. This file is retained as a compatibility pointer because earlier presentation notes referenced it.

The current implementation includes Gemini image triage, Open-Meteo weather context, persisted draft/analysis/report state, manual final-severity selection, private local evidence storage, and persisted map markers. The current implementation does not include an AI queue, signed URLs, object storage, scheduled draft cleanup, automatic AI retries/backoff, idempotency keys, duplicate rejection, malware scanning, or official verification.
